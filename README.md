## :bookmark: Sobre

O <strong>WeatherPlay</strong> é uma aplicação Web capaz de recomendar playlists de acordo com o clima atual da sua cidade.

<a id="tecnologias-utilizadas"></a>

## :rocket: Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias

- [TypeScript](https://www.typescriptlang.org/)
- [ReactJS](https://reactjs.org/)
- [Chakra](https://chakra-ui.com/)
- [Open Weather Map](https://openweathermap.org/)
- [RapidApi Shazam](https://rapidapi.com/apidojo/api/shazam)

<a id="como-usar"></a>

## :fire: Como usar

- ### **Pré-requisitos**

  - É **necessário** possuir o **[Node.js](https://nodejs.org/en/)** instalado na máquina
  - Também, é **preciso** ter um gerenciador de pacotes seja o **[NPM](https://www.npmjs.com/)** ou **[Yarn](https://yarnpkg.com/)**.

1. Faça um clone :

```sh
  $ git clone https://github.com/williamroberttv/weatherplay.git
```

2. Executando a Aplicação:

```sh
  # Instale as dependências
  $ npm install
  ou
  $ yarn
  $ yarn install

  # Inicie a aplicação web
  $ npm start
  ou
  $ yarn start

```
